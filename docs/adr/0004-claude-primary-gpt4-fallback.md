# ADR-0004: Claude Haiku как основной LLM, GPT-4 как fallback

- **Статус:** Accepted
- **Дата:** 2026-04-24
- **Авторы:** команда Telegram Lead Aggregator

## Контекст

LLM-классификация — самая дорогая часть конвейера (ТЗ FEATURE-05). Ожидаемый объём — до 10 000 сообщений/день, из которых ~5–10% проходят keyword-фильтр и попадают в LLM. При целевой цене < $0.005 за анализ (ТЗ FEATURE-05) и бюджете $80–150/мес на API (ТЗ разд. 5) — это ~20 000–30 000 LLM-вызовов в месяц.

Качество на русском языке важно: целевая аудитория — СНГ-сообщества, большая часть сообщений не на английском.

## Решение

- **Основной провайдер:** Anthropic Claude Haiku (актуальная версия на момент релиза). Оправдание: лучшее качество на русском среди дешёвых моделей, стоимость укладывается в $0.005/сообщение при типичных ~1–2K токенов контекста.
- **Fallback:** OpenAI GPT-4 (gpt-4o или актуальный turbo-класс). Включается только при недоступности Anthropic API (5xx, таймауты, rate-limit). Не используется параллельно.
- **Бюджетный cap:** жёсткий лимит `LLM_COST_CAP_USD_PER_MESSAGE=0.005`. Превышение → сообщение помечается `status=error`, алерт в Sentry. Дневной cap (`LLM_DAILY_BUDGET_USD`) — circuit breaker для worker'а.
- **Версионирование промптов** — через `PROMPT_VERSION` env + каталог `prompts/vN/`, сохраняется в `lead_analysis.prompt_version` (см. `docs/prompts-versioning.md`).

## Рассмотренные альтернативы

- **Claude Sonnet основной.** Качество выше, но цена ~5x — не укладывается в бюджет.
- **GPT-4 основной.** Дороже Haiku в 2–3x при сопоставимом качестве на русском.
- **Локальная модель (Llama, Qwen).** Качество классификации на русском ниже + операционные затраты на GPU-хостинг превышают экономию.

## Последствия

- **Позитивные.** Предсказуемый OPEX; fallback защищает SLA при инцидентах Anthropic.
- **Негативные.** Двойная интеграция (Claude SDK + OpenAI SDK) и double-quality-assurance промптов.
- **Операционные.** При cost-spike применяется runbook `llm-cost-spike.md`; ручная валидация precision раз в 2 недели (ТЗ FEATURE-05).
